Beyond Waste

Beyond Waste is a full-stack food redistribution platform designed to reduce food waste by connecting food donors with NGOs, community organizations, and delivery partners. The platform streamlines the donation process through secure user authentication, role-based access control, and an intuitive workflow for managing food donations from creation to successful distribution.

Built using Java, Spring Boot, React, and MySQL, the application follows the MVC architecture and RESTful API principles while providing a scalable, maintainable, and responsive solution for real-world food redistribution.

Key Features
Multi-role user management (Donors, NGOs, Receivers, Delivery Partners, Admin)
Secure authentication and role-based authorization
Food donation lifecycle management
RESTful API architecture
Responsive user interface
Normalized relational database design
Admin dashboard for platform management
Git-based version control
Scalable MVC architecture
Tech Stack
Backend: Java, Spring Boot, Hibernate, JPA
Frontend: React, HTML, CSS, JavaScript, Bootstrap
Database: MySQL
Tools: Git, GitHub
